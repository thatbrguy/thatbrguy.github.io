Hi! I am Bharath Raj

I currently work at Siemens Digital Industries Software as part of the Intelligent Controls Systems (ICS) team where I build models, algorithms and systems for myriad autonomous driving and general machine learning applications.

I am generally interested in various problems related to perception and machine learning. Of late, I am specifically interested in researching about how to best structure learning-based methods with traditional algorithms and theory from geometry and physics to create perception systems that can better handle the challenges of complex real world scenarios.

I am always on the lookout for opportunities to learn more and work on exciting projects in the fields of my interest. Scroll down to take a look at some of my projects, publications and blogs.

Project

NeRF TensorFlow 2

  • Created an implementation of NeRF from scratch in TensorFlow 2 for 360-degree inward-facing scenes.
  • Includes implementation of components such as an algorithm to solve for the nearest 3D point to N lines, inverse transform sampling and more.
  • The implementation is available at this GitHub repository.
Project

Open Source Contributions to Kornia

  • Implemented a Direct Linear Transform (DLT) based Perspective-n-Point (PnP) solver in PyTorch.
  • Contributed enhancements and fixes to various parts of the repository as well.
Publication

Exploring Techniques to Improve Activity Recognition using Human Pose Skeletons

Bharath Raj N., Anand Subramanian, Kashyap Ravichandran, Venkateswaran N.

  • Explored the efficacy of using hand crafted feature extraction techniques and some train-time techniques such as keypoint dropout on improving human pose skeleton based activity recognition performance.
  • Published at the HADCV workshop at WACV 2020.
  • You can checkout the paper here.
Publication

Single Image Haze Removal Using a Generative Adversarial Network

Bharath Raj N., Venkateswaran N.

  • Created a conditional GAN (cGAN) based architecture to dehaze images. The generator of the cGAN used the 56 Layer Tiramisu model. A weighted loss function was used for the generator.
  • Code and first version of the preprint were launched in 2018. Project currently has more than 70 stars on GitHub.
  • Paper published at WiSPNET 2020.
  • Relevant links: [GitHub | arXiv]
